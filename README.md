# novelty calculator
> Calculate the novelty based on your location.

## Usage
> How to use it:
```python
from novelty import most_novel_angle


angle = most_novel_angle(
    40,  # your x
    120,  # your y
    cities=[  # city coordinates
        [0, 0],
        [120, 0],
        [200, 325],
        [100, 55]
    ]
)
```
